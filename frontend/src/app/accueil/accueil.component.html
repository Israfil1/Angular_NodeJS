BACK-END

  • NodeJS : NodeJS est un environnement serveur basé sur le langage JavaScript.
  • nodemon : Utilitaire d'interface de ligne de commande (CLI) qui enveloppe NodeJS et qui surveille le système de fichiers et redémarre automatiquement le processus.
  • WAMP: Gestion serveur MYSQL et PHP, nous utliserons également PhpMyAdmin pour gérer notre base de données
  • Express: Infrastructure d'application (framework) en JavaScript hébergée dans l'énvironnement d'éxécution Nodejs. Utilisation du Router pour l'architecture d'acccès aux données
  • MySQL: Base de données en langage SQL, définition et manipulation via des requêtes envoyées depuis le serveur

FRONT-END

  • Angular : Framework JavaScript open source basé sur Typescript
  • Bootstrap Framework utilisant des langages HTML CSS & JavaScript
